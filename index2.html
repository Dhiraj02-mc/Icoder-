🌐 HTML Tutorial for Beginners

📌 What is HTML?

HTML (HyperText Markup Language) is the standard language used to create webpages. It provides the basic structure of web pages using tags like <h1>, <p>, <a>, etc.

🔧 Basic HTML Structure

Here’s the basic skeleton of an HTML document:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

⭐ Common HTML Tags

  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <a href> – Link
  • <img src> – Image
  • <ul>, <ol>, <li> – Lists
  • <div> – Division or section

💡 Example: A Simple Webpage

<!DOCTYPE html>
<html>
  <head>
    <title>My First Site</title>
  </head>
  <body>
    <h1>Welcome to iCoder</h1>
    <p>Learning HTML is fun and easy!</p>
    <a href="https://google.com">Visit Google</a>
  </body>
</html>
🎉 You're ready to start coding in HTML!
